### 5.4 Validation Rules

**Per-Root Validation:**
```rust
fn validate_root(r: &[f32; 8]) -> Result<(), E8Error> {
    // Check norm-squared = 2
    let norm_sq: f32 = r.iter().map(|&x| x * x).sum();
    if (norm_sq - 2.0).abs() > 1e-6 {
        return Err(E8Error::InvalidNorm(norm_sq));
    }
    
    // Check Type I or Type II structure
    let is_type_i = r.iter().filter(|&&x| x != 0.0).count() == 2
                    && r.iter().all(|&x| x.abs() == 1.0 || x == 0.0);
    
    let is_type_ii = r.iter().all(|&x| x.abs() == 0.5)
                     && (r.iter().filter(|&&x| x < 0.0).count() % 2 == 0);
    
    if !(is_type_i || is_type_ii) {
        return Err(E8Error::InvalidRootStructure);
    }
    
    Ok(())
}
```

```rust
        let coords = experience_to_coords(exp);
        let nearest = find_nearest_root(&coords, roots);
        *weights.entry(nearest).or_insert(0.0) += 1.0;
    }
    
    // Normalize weights
    let total: f32 = weights.values().sum();
    for w in weights.values_mut() {
        *w /= total;
    }
    
    // Compute barycenter
    let mut centroid = [0.0; 8];
    for (&idx, &weight) in &weights {
        for i in 0..8 {
            centroid[i] += weight * roots[idx][i];
        }
    }
    
    centroid
}
```

**Step 4: Nearest Root Quantization**
```rust
fn find_nearest_root(point: &[f32; 8], roots: &[Root]) -> usize {
    roots.iter()
         .enumerate()
         .min_by(|(_, a), (_, b)| {
             let dist_a = euclidean_distance(point, a);
             let dist_b = euclidean_distance(point, b);
             dist_a.partial_cmp(&dist_b).unwrap()
         })
         .map(|(idx, _)| idx)
         .unwrap()
}
```

**Tie-Breaking Rule:** If multiple roots are equidistant, select the one with the lowest index.

## 8. Graph Model

### 8.1 Adjacency Rules

Two roots Œ±, Œ≤ are **adjacent** iff `|‚ü®Œ±, Œ≤‚ü©| = 1`.

**Construction:**
```rust
fn build_adjacency(roots: &[Root; 240]) -> Vec<Vec<usize>> {
    let mut adj = vec![vec![]; 240];
    
    for i in 0..240 {
        for j in (i+1)..240 {
            let ip = inner_product(&roots[i], &roots[j]);
            if (ip.abs() - 1.0).abs() < 1e-6 {
                adj[i].push(j);
                adj[j].push(i);
            }
        }
    }
    
    adj
}
```

**Expected Degree:** Each root has exactly **56 neighbors**.

### 8.2 Data Layout

```rust
struct E8Graph {

}

type Root = [f32; 8];
```

## 9. Algorithms

### 9.1 Root-to-Domain Affinity

**Inputs:**
- `root: &[f32; 8]`
- `domain: &[f32; 8]` (unit-norm)

**Output:**
- `affinity: f32` ‚àà [-1, 1]

**Algorithm:**
```rust
fn affinity(root: &[f32; 8], domain: &[f32; 8]) -> f32 {
    let ip: f32 = root.iter().zip(domain.iter())
                      .map(|(r, d)| r * d)
                      .sum();
    let root_norm: f32 = root.iter().map(|x| x * x).sum::<f32>().sqrt();
    ip / root_norm
}
```

**Complexity:** O(8) = O(1)

**Failure Modes:**
- `NearZeroVector` if `root_norm < 1e-8`

**Validation:**
- Assert `|domain_norm - 1.0| < 1e-6`

### 9.2 Multi-Domain Projection

**Inputs:**
- `point: &[f32; 8]`
- `domains: &[DomainVector]`

**Output:**
- `DomainProjection { weights: HashMap<String, f32> }`

**Algorithm:**
```rust
fn project_to_domains(
    point: &[f32; 8],
    domains: &[DomainVector],
) -> DomainProjection {
    let mut weights = HashMap::new();
    for domain in domains {
        let aff = affinity(point, &domain.normalized);
        weights.insert(domain.label.clone(), aff);
    }
    DomainProjection { weights }
}
```

**Complexity:** O(D) where D = number of domains

### 9.3 Trajectory Construction

**Inputs:**
- `start: usize` (root index)
- `goal: usize` (root index)
- `momentum: &[f32; 8]`
- `attractors: &[(usize, f32)]` (root index, strength)
- `graph: &E8Graph`
- `max_steps: usize`

**Output:**
- `Trajectory { path: Vec<usize>, timestamps: Vec<f64> }`

**Algorithm:**
```rust
fn construct_trajectory(
    start: usize,
    goal: usize,
    momentum: &[f32; 8],
    attractors: &[(usize, f32)],
    graph: &E8Graph,
    max_steps: usize,
) -> Trajectory {
    let mut path = vec![start];
    let mut current = start;
    
    for step in 0..max_steps {
        if current == goal {
            break;
        }
        
        // Score all neighbors
        let neighbors = &graph.adjacency[current];
        let next = neighbors.iter()
                            .map(|&idx| {
                                let jump_cost = euclidean_distance(
                                    &graph.roots[current],
                                    &graph.roots[idx]
                                );
                                let momentum_alignment = inner_product(
                                    &graph.roots[idx],
                                    momentum
                                );
                                let attractor_force: f32 = attractors.iter()
                                    .map(|&(a_idx, strength)| {
                                        strength / (1.0 + euclidean_distance(
                                            &graph.roots[idx],
                                            &graph.roots[a_idx]
                                        ))
                                    })
                                    .sum();
                                
                                let score = -jump_cost 
                                          + 0.5 * momentum_alignment
                                          + 0.3 * attractor_force;
                                (idx, score)
                            })
                            .max_by(|(_, s1), (_, s2)| {
                                s1.partial_cmp(s2).unwrap()
                            })
                            .map(|(idx, _)| idx)
                            .unwrap();
        
        path.push(next);
        current = next;
    }
    
    Trajectory {
        id: Xuid::generate(),
        path,
        timestamps: vec![],
        metadata: HashMap::new(),
    }
}
```

**Complexity:** O(max_steps √ó degree √ó attractors) = O(max_steps √ó 56 √ó A)

**Parameters:**
- `momentum_weight: 0.5` (tunable)
- `attractor_weight: 0.3` (tunable)
- `max_steps: 1000` (default)

**Failure Modes:**
- `MaxStepsExceeded` if goal not reached
